# Pinboard Login, Move-Menüs und Board-Toolbar Refinement

Diese Version setzt die nächste Optimierungsrunde um:

- neuer Login-Screen mit SVG-/Bild-artigem Hintergrund, Glas-Blur-Panel, Passwort-Sichtbarkeit und Google-Login-Button
- Hauptseite ohne leuchtendes Icon und ohne „Pinboards“-Branding oben links
- höher gesetzter Editorial-Hero für Private/Business Pinboards
- cleanere Hauptseiten-Toolbar ohne gemeinsamen Außenrahmen
- Boards können per Drag-Handle in andere Board-Bereiche verschoben werden
- Boards können per Rechtsklick über ein Move-Dialog-Menü zwischen Bereichen sowie zwischen Private und Business verschoben werden
- Pins können per Rechtsklick über ein Move-Dialog-Menü in andere Teilbereiche verschoben werden
- Theme-Switch zeigt im Darkmode eine Sonne und im Lightmode einen Mond
- Board-Cover-Verläufe wurden weicher gestaltet, ohne harten schwarzen Strich
- Rundungen wurden weiter reduziert und konsistenter angewendet

`npm run build` wurde erfolgreich ausgeführt.
